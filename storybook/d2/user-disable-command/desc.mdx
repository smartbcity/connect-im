


### Command

Type: `UserDisableCommand`

---

***id*** [`UserId`](#userid)

Identifier of the user to disable.



---

***disabledBy*** [`UserId?`](#userid)

Identifier of the user executing the command.



---

***anonymize*** `Boolean`

Whether to anonymize the personal data of the user or not. This will remove or blank all sensible fields except custom attributes. To anonymize custom attributes, see `attributes` field.



---

***attributes*** `Map<String, String>?`

Custom attributes to update during anonymization. Use this field only if `anonymize` is set to true.
